Bugs

So, you found a bug?

  • If you're not using the latest released version, please upgrade and see if the bug has already been fixed.
  • Check existing bug reports to see if the bug has already been reported.

To report a bug or request a feature, or maybe contribute with a patch, you may:

  • Report an issue on GitHub.
  • Submit pull requests on GitHub.
  • Discuss in GitHub discussions.
  • Discuss on the ccache mailing list.
  • You can also send patches directly to the ccache maintainer (Joel Rosdahl) if you like.
